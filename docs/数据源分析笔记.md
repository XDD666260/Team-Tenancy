## 二、逐站分析

### 1. 链家 (cq.lianjia.com) — 🔥主数据源

**列表页 URL 规律：**
```
首页：       https://cq.lianjia.com/ershoufang/
分页：       https://cq.lianjia.com/ershoufang/pg2/   ... pg100/
区县筛选：   https://cq.lianjia.com/ershoufang/yubei/pg1/
朝向筛选：   f1东 f2南 f3西 f4北 f5南北
楼层筛选：   lc1低楼层 lc2中楼层 lc3高楼层
户型筛选：   l1一室 l2二室 l3三室 ... l6五室以上
组合筛选：   https://cq.lianjia.com/ershoufang/yubei/f5lc2l3/
详情页：     https://cq.lianjia.com/ershoufang/106121336064.html
特殊区域：   https://cq.lianjia.com/ershoufang/liangjiangxinqu/ （两江新区）
```

**数据格式**：HTML 页面

**关键CSS选择器（已通过Console验证 ✅）：**

| 字段 | CSS选择器 | 验证结果 | 示例值 |
|------|----------|---------|--------|
| 房源标题 | `.title a` | ✅ | "龙湖春森彼岸 3室2厅" |
| 总价 | `.totalPrice span` | ✅ | "39万" |
| 单价 | `.unitPrice span` | ✅ | "9,275元/平" |
| 小区名称 | `.positionInfo a:first-child` | ✅ | "茂业大厦" |
| 商圈 | `.positionInfo a:nth-child(2)` | ✅ | "观音桥" |
| 户型/面积/朝向/装修/楼层 | `.houseInfo` | ✅ | "1室0厅 \| 42.05平米 \| 东南 \| 精装 \| 中楼层" |
| 关注人数 | `.followInfo` | ✅ | "128人关注" |
| 发布时间 | `.followInfo` | ✅ | "6天前发布" |

> ⚠️ 区县信息不在列表页HTML中，从爬取的 URL 里提取：`/yubei/` → "渝北区"

**houseInfo 字段解析规则：**
```
原始字符串: "1室0厅 | 42.05平米 | 东南 | 精装 | 中楼层(共32层)"
按 " | " 分割:
  [0] "1室0厅"   → 正则提取 rooms=1, halls=0
  [1] "42.05平米" → 替换"平米"后 astype float → area
  [2] "东南"      → orientation
  [3] "精装"      → decoration  
  [4] "中楼层(共32层)" → 正则提取 floor_type="中层", total_floors=32
```

**详情页补充字段：**

| 字段 | 所在位置 | 验证 |
|------|---------|------|
| 经度/纬度 | 开发者工具-网络-抓包-解析API返回的JSON数据 | 验证 |
| 建成年代 | 详情页小区信息区域 xiaoqu_main f1   xiaoqu_info| 验证 |
| 详细地址 | 详情页 `.areaName` | ☐ 待验证 没有找到 |

**反爬措施：**
- 需要登录：是（浏览不需要，大量爬取可能需要）
- 人机验证：有
- 需要 Cookie：是
- 封IP风险：中等

**爬取策略：**
- 每次请求间隔：**1.5~3秒**随机
- 并发数：**5~8线程**（不宜太高，链家反爬敏感）
- 需要 User-Agent 轮换：是（准备5个真实UA）
- 需要代理池：视情况（如果频繁弹验证码则需要）

分页：https://chongqing.anjuke.com/sale/yubei/p6/?from=HomePage_TopBar

每页 **60条**，同区县翻页到50页即3000条。

## 三、最终选型决策

### 主数据源：链家

| 项目 | 详情 |
|------|------|
| 网站 | https://cq.lianjia.com/ershoufang/ |
| 数据格式 | HTML |
| 每页条数 | 30条 |
| 预计总量 | 主城9区 × 3000条（100页）+ 其他区县 × 1500条 ≈ **5~6万条** |
| CSS选择器 | ✅ 已全部验证通过 |
| 测试爬虫 | ☐ 待写 |

### 备用数据源：安居客

| 项目 | 详情 |
|------|------|
| 网站 | https://chongqing.anjuke.com/sale/ |
| 数据格式 | HTML |
| 每页条数 | 60条（链家的2倍） |
| 预计总量 | 链家不够时补充主要区县，每区县3000条 |
| CSS选择器 | ☐ 待验证 |
| 测试爬虫 | ☐ 待写 |

### 选型理由

1. **链家数据质量最高**：房源信息最全，字段规范统一，覆盖重庆所有区县，CSS选择器已经Console验证通过
2. **贝壳与链家数据同源**（都是贝壳集团），爬了链家再爬贝壳数据大量重复，且贝壳反爬更严，不值得
3. **安居客作为补充**：数据独立于链家体系，反爬门槛低（无需登录），每页60条效率高，链家数据不够5万时快速补量
4. **房天下/58同城不采用**：数据质量参差不齐，虚假房源多，优先保证链家+安居客达标
