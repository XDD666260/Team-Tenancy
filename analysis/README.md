# 算法分析模块

## 功能概述

本模块实现了重庆二手房数据的完整分析流水线：

### 1. 房价预测模型
- **RandomForest** 随机森林回归器（200棵树，max_depth=15）
- **GradientBoosting** 梯度提升回归器（200棵树，max_depth=6）
- 目标变量：总价（万）和 单价（元/㎡）
- 特征：面积、户型、楼层、房龄、装修、朝向、区县编码、小区编码等12个特征

### 2. 预测误差指标

| 模型 | 目标 | 训练R² | 测试R² | 测试MAE | 测试RMSE | 5折CV R² |
|------|------|--------|--------|---------|----------|----------|
| RandomForest | 总价 | 0.8467 | 0.6917 | 16.53万 | 31.05万 | 0.6938±0.04 |
| RandomForest | 单价 | 0.7858 | 0.6241 | 1491元/㎡ | 2427元/㎡ | 0.6182±0.02 |
| GradientBoosting | 总价 | 0.9454 | 0.6981 | 16.13万 | 30.72万 | 0.6963±0.06 |
| GradientBoosting | 单价 | 0.8830 | 0.6294 | 1492元/㎡ | 2410元/㎡ | 0.5970±0.02 |

### 3. 特征重要性排名（TOP 5）

| 排名 | 总价预测 | 单价预测 |
|------|----------|----------|
| 1 | 面积 (0.558) | 小区均价编码 (0.721) |
| 2 | 小区均价编码 (0.291) | 面积 (0.077) |
| 3 | 户均面积 (0.038) | 户均面积 (0.064) |
| 4 | 总楼层 (0.030) | 总楼层 (0.042) |
| 5 | 房龄 (0.027) | 卫 (0.023) |

### 4. KMeans 聚类分析（5类）
- 聚类0：低价小户型 (7915元/㎡, 64.6㎡) — 17.4%
- 聚类1：中等价位主力户型 (9687元/㎡, 94.1㎡) — 36.7%
- 聚类2：老房低价大户型 (3641元/㎡, 113.8㎡) — 18.3%
- 聚类3：次新房中等户型 (5458元/㎡, 95.6㎡) — 19.9%
- 聚类4：高端大户型 (12644元/㎡, 162.6㎡) — 7.8%

轮廓系数: 0.1461

## 快速开始

```bash
# 运行完整分析流水线
python -m analysis.run_analysis
```

## 输出文件

- **模型**: `analysis/output/models/` — models.pkl, scaler.pkl, kmeans.pkl, feature_cols.pkl
- **图表**: `analysis/output/charts/` — 15张PNG图表
  - feature_importance_*.png (6张) — 特征重要性排名
  - prediction_*.png (4张) — 预测效果散点图+残差图
  - clustering_*.png (4张) — 聚类肘部法则、PCA、雷达图、区县分布
  - district_price_ranking.png (1张) — 区县价格排名
- **数据库**: 结果自动写入 `analysis_results` 表
